## Multi-Agent Assistant

The assistant automatically routes questions to specialized agents:

- Schema Agent
- Validation Agent
- Data Analysis Agent
- Normalization Agent
- SQL Agent
- GTN Agent
- General Migration Agent

Example conversation:

User: What type is score?
Agent: Schema Agent

User: Should username be the primary key?
Agent: Data Analysis Agent

User: Why was this normalization proposed?
Agent: Normalization Agent
